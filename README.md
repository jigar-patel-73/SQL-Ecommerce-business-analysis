# E-commerce Sales & Customer Analytics | SQL Project

Advanced SQL portfolio project analyzing e-commerce customers, revenue, rankings, trends, and business insights using 25 real-world SQL queries.

## Skills Demonstrated

* SQL Joins
* Aggregations
* Subqueries
* CTEs
* Window Functions
* RANK / PERCENT_RANK
* LAG / LEAD
* Running Totals
* Business KPI Analysis

## Project Structure

* `Ecommerce_SQL_Master_Project.sql`

## Key Business Questions Solved

* Top customers by spending
* Revenue by city / country
* Monthly revenue trends
* Repeat customers
* Top 25% customers
* Top spenders by country
* Customer behavior over time
